Внутрішній сервіс транскрибації дзвінків

Командний проєкт, виконаний під час проходження практики

Опис проєкту

У межах проходження практики було розроблено внутрішній сервіс для запису та транскрипції дзвінків у Google Meet. Система дає змогу записувати аудіо, передавати його на сервер, виконувати автоматичну транскрипцію мовлення, розділяти мовців та зберігати результати у Google Drive.

Мета роботи

Основною метою роботи було створення сервісу, який автоматизує обробку аудіо з онлайн-зустрічей та дозволяє отримувати текстову транскрипцію розмови для подальшого використання.

Використані технології

  • JavaScript — для реалізації клієнтської частини у вигляді Chrome Extension.
  • Node.js — для створення серверної частини та обробки HTTP-запитів.
  • Python — для реалізації модуля транскрипції аудіо.
  • Faster-Whisper — для автоматичного розпізнавання мовлення.
  • pyannote.audio — для розділення мовців.
  • Google APIs / OAuth 2.0 — для авторизації та завантаження результатів у Google Drive.

Етапи виконання роботи

  • Аналіз поставленого завдання та формування структури системи.
  • Розробка клієнтської частини для запису аудіо з Google Meet.
  • Створення серверної частини для прийому та обробки аудіофайлів.
  • Інтеграція Faster-Whisper для транскрипції мовлення.
  • Інтеграція pyannote.audio для розділення мовців.
  • Налаштування збереження результатів у Google Drive.
  • Тестування повного циклу роботи системи.
